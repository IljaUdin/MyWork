APU.exe - ПО для работы с антенно-поворотным устройством.

UpdateFirmware.exe - ПО для загрузки прошивки. 
При обновлении прошивки должен находиться в той же папке, где и основной файл программы (APU.exe).
В остальных случаях его отсутствие не влияет на корректность работы ПО.
При загрузке новой прошивки (файл с расширением .bin) на АПУ файл прошивки должен находиться в 
той же папке, где и файл программы.
